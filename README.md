# AI Powered Assistive Mobility Tool for Visually Impaired

This project was developed during my Infosys Springboard Internship. The goal of the project is to assist visually impaired individuals in safe navigation using AI-based technologies.

## Project Objective
To design an assistive system that helps visually impaired users detect obstacles and navigate their surroundings using AI and real-time processing.

## Features
- Obstacle detection using AI
- Real-time data processing
- Assistive feedback for navigation
- User-friendly interface

## Technologies Used
- JavaScript / TypeScript
- React
- Supabase
- Node.js
- AI integration modules
